**=======================================
PENGGUNAAN MENTION MODERATOR DISCORD
=======================================**

Untuk menghindari role ping abuse, tag @Moderator Discord hanya boleh digunakan **JIKA DAN HANYA JIKA ketika ada kejadian luar biasa/urgensi tinggi, atau rule violation.** Selebihnya, jika ingin tag mod, tag salah satu dari Admin atau Moderator yang sedang online.

Menggunakan role tag @Admin dan atau @Moderator Discord di luar hal tersebut, Anda akan dikenai sanksi yang berlaku di server ini.

Kejadian luar biasa atau urgensi tinggi dalam hal ini adalah:

1. Spam secara masif di salah satu/beberapa channel.
2. Doxxing (ancaman pembeberan informasi pribadi terhadap salah satu/beberapa member di sini).
3. Berbagi konten vulgar/NSFW.
4. Serangan akun bot yang menyertakan gambar tak pantas maupun yang dapat membuat kengerian/ketakutan.

Apabila ada kejadian di luar yang disebutkan di atas, **maka dianjurkan untuk mention salah satu Admin atau Moderator yang ada**. 

**=======================================
ATURAN MENGENAI PROFILE PICTURE, USERNAME, DAN/ATAU NICKNAME
=======================================**

Mengingat kembali bahwa server ini adalah family friendly, maka akan diberlakukan aturan mengenai penggunaan profile picture, username, dan nickname.

Profile picture, username dan nickname diwajibkan untuk tidak mengandung unsur:

- SARA
- Politik
- Genital/alat kelamin (secara langsung maupun tak langsung)
- Swear words/kata-kata kasar
- Trolling (yang bisa memancing orang untuk marah/tersinggung)

Apabila didapati hal di atas, maka kami akan meminta kamu untuk mengganti profile picture, username, dan/atau nickname terlebih dahulu. Kalau menolak dan tetap bersikeras, maka kami akan mencabut role "Verified Member" kamu yang mengakibatkan kamu tidak bisa mengakses keseluruhan isi server ini.

**=======================================
ATURAN MENGENAI AKUN GANDA (2ND ACCOUNT)
=======================================**

Penggunaan akun ganda (2nd account) di server ini **SANGAT TIDAK DIPERBOLEHKAN**. Namun, apabila Anda **sangat terpaksa** harus menggunakan akun ganda di server ini, maka setelah masuk ke server Anda **wajib melaporkannya** kepada salah satu Admin dan atau Moderator Discord dengan menyertakan alasan yang bisa diterima melalui DM.

Alasan yang bisa diterima perihal penggunaan akun ganda (akan diperbarui sesuai kebutuhan):
- Anda tidak bisa mengakses akun utama karena lupa kata sandi (password) dan surel (e-mail) sehingga tidak bisa memperoleh kembali akun tersebut.
- Kerusakan maupun kehilangan gawai (gadget) yang mengakibatkan akun utama Anda tidak bisa diperoleh kembali dengan cara apapun.

Meskipun demikian, hal tersebut bukan menjadi alasan untuk bisa masuk ke server ini begitu saja. Admin dan Moderator **berhak** menanyakan secara rinci dan mengusahakan Anda untuk tetap menggunakan akun utama apabila memungkinkan.

**PEMBLOKIRAN/BAN**

Apabila Admin atau Moderator menemukan dugaan adanya akun ganda tanpa pelaporan terlebih dahulu di server ini, mereka berhak untuk memanggil terduga pemilik akun tersebut untuk ditanyai lebih lanjut. Kalau terduga terbukti dan tidak bisa diajak kooperatif (bekerja sama) maka akun ganda beserta pemiliknya akan dikeluarkan (juga ban) dari server ini.

**=======================================
ATURAN MENGENAI DRAMA
=======================================**

Berhubung akhir-akhir ini banyak drama yang terjadi di ranah VTuber dan sekitarnya, **maka kami meminta kerja sama Anda untuk tidak membahas (sekaligus berdebat) drama tersebut dalam bentuk apapun di server ini.**

Mengapa kami mengeluarkan aturan tersebut?
**- Meminimalisasi perdebatan yang terjadi di server ini.
- Tidak menimbulkan pandangan negatif terhadap pihak-pihak terkait.
- Membuat Server agar tetap kondusif.**

Kami tidak mempermasalahkan apabila Anda membahas suatu drama dalam lingkar diskusi secara tertutup (misalnya melalui DM grup Discord), tetapi tidak diizinkan untuk di server ini. Sekadar mengingatkan bahwa server ini adalah family friendly, jadi ada baiknya kita semua mematuhi hal tersebut.

**=======================================
ATURAN MENGENAI BERITA HOAX
=======================================**

Kami menghargai hak setiap individu untuk berpendapat dan berekspresi, namun pada saat yang sama kami juga menghormati kebenaran dan kejujuran. Oleh karena itu, kami melarang penyebaran berita hoax atau informasi yang tidak akurat di server Discord ini. Kami akan mengambil tindakan sesuai dengan peraturan yang berlaku jika terdapat laporan atau bukti penyebaran berita hoax. Kami berharap setiap member server dapat berperan dalam menjaga kredibilitas dan integritas informasi di server ini.

**=======================================
PEMBERIAN SANKSI
=======================================**

Pada dasarnya apabila rekan-rekan melanggar hal-hal yang telah disebutkan di atas, Moderator Discord untuk berhak memberikan teguran agar yang bersangkutan mengetahui apa yang harus diperbaiki / diubah.
Namun bilamana member tersebut tidak mengindahkan "salam hangat" dari Moderator Discord dan masih melakukan pelanggaran, maka kami akan lakukan tindakan sebagai berikut:

- 1 kali: mute selama 1 jam
- 2 kali: mute selama 1 hari
- 3 kali: mohon maaf, Anda akan mendapatkan "surat cinta" dari Moderator Discord.

Dengan ini, kami menegaskan bahwa setiap member yang bergabung di server Discord ini sudah membaca, mengerti, dan akan selalu menaati peraturan yang berlaku di server ini. Kami juga mengingatkan bahwa setiap pelanggaran akan dikenakan sanksi sesuai dengan ketentuan yang telah ditetapkan. Kami berharap, lingkungan server Discord ini dapat tetap nyaman dan kondusif untuk seluruh member server.
